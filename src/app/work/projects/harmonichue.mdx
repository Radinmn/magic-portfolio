## Overview

A creative web experience that connects sound, color, and interaction. Users move through a guided interface and generate a personalized visual output based on their input.

## What I worked on

- Built the experience using HTML, CSS, JavaScript, and p5.js
- Designed interactive visual feedback and animated transitions
- Structured the user flow to keep the experience clear and engaging
- Refined the interface based on user testing and feedback

## Why it stands out

HarmonicHue turns creative coding into an immersive web experience instead of a static webpage.
