## Overview

A creative web experience that connects sound, color, and interaction. Users move through a guided interface and generate a personalized visual experience based on their input.

## What I worked on

- Built the experience using HTML, CSS, JavaScript, and p5.js
- Designed interactive visual feedback and animated transitions
- Structured the user flow to keep the experience clear and engaging
- Refined the interface based on user testing and feedback

## Why it stands out

HarmonicHue shows how creative coding can be used to make web experiences feel expressive, immersive, and memorable instead of static and conventional.
